# Weather Dashboard

## Description
A responsive Weather Dashboard built using HTML, CSS, and JavaScript. It fetches real-time weather information using the OpenWeatherMap API.

## Features
- Search weather by city
- Displays temperature
- Displays humidity
- Displays wind speed
- Shows weather description
- Uses Fetch API and async/await
- Handles invalid city names gracefully

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Fetch API
- OpenWeatherMap API

## How to Run
1. Download or clone the project.
2. Open the project in Visual Studio Code.
3. Replace the API key in `script.js` with your own OpenWeatherMap API key.
4. Open `index.html` using Live Server.
5. Enter a city name and click **Search**.
